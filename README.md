# Multi-Modal-Sensing-Data-Based-Real-Time-Path-Loss-Prediction-for-6G-UAV-to-Ground-Communications
M. Sun, L. Bai, Z. Huang, and X. Cheng, “Multi-modal sensing data based real-time path loss prediction for 6G UAV-to-ground communications,” IEEE Wireless Communications Letters, vol. 13, no. 9, pp. 2462-2466, Sept. 2024. 

## Dependencies and Installation
- Python 3.9 (Recommend to use [Anaconda](https://www.anaconda.com/))
- Pytorch 2.0.0
- NVIDIA GPU/CPU + CUDA 
- Python packages: `pip install -r requirements.txt`

### Inference command 
```
python test_image_pl.py
```
